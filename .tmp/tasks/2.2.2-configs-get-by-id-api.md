## 概要
特定の設定項目を取得するAPIを実装する。config_passwordによる認証後に指定したキーの設定項目詳細を取得できる機能を提供する。

## 実装内容

### APIエンドポイント
- **URL**: `GET /api/v1/configs/:key`
- **認証**: 必須（config_password認証）
- **レスポンス形式**: JSON

### パスパラメータ
- `key`: 設定項目のキー（例: "google.client_id"）

### レスポンス仕様
```typescript
interface ConfigResponse {
  success: boolean;
  data: {
    key: string;
    value: string;
    type: 'string' | 'boolean' | 'number' | 'json';
    description: string | null;
    system_config: boolean;
    validation: ValidationRule | null;
    created_at: string;       // ISO 8601形式
    updated_at: string;       // ISO 8601形式
  };
  message?: string;
}

interface ValidationRule {
  required?: boolean;
  type?: 'string' | 'boolean' | 'number' | 'json';
  minLength?: number;
  maxLength?: number;
  min?: number;
  max?: number;
  pattern?: string;
  enum?: string[];
  errorMessage: string;
  default?: string | number | boolean;
}
```

### エラーレスポンス
```typescript
interface ConfigErrorResponse {
  success: false;
  error: {
    code: 'UNAUTHORIZED' | 'FORBIDDEN' | 'NOT_FOUND' | 'VALIDATION_ERROR' | 'INTERNAL_ERROR';
    message: string;
    details?: Record<string, string>;
  };
}
```

- 400: 不正なキーフォーマット
- 401: 認証エラー
- 403: 権限不足
- 404: 設定項目が見つからない
- 500: サーバーエラー
