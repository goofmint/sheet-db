## 概要
特定の設定項目を取得するAPIを実装する。認証済みユーザーが指定したIDの設定項目詳細を取得できる機能を提供する。

## 実装内容

### APIエンドポイント
- **URL**: `GET /api/v1/configs/:id`
- **認証**: 必須（JWT認証）
- **レスポンス形式**: JSON

### パスパラメータ
- `id`: 設定項目のUUID

### レスポンス仕様
```typescript
interface ConfigResponse {
  id: string;
  key: string;
  value: string;
  description?: string;
  type: 'string' | 'number' | 'boolean' | 'json';
  isSystem: boolean;
  createdAt: string;
  updatedAt: string;
  metadata?: {
    validationRules?: object;
    displayOptions?: object;
  };
}
```

### エラーレスポンス
- 400: 不正なIDフォーマット
- 401: 認証エラー
- 403: 権限不足
- 404: 設定項目が見つからない
- 500: サーバーエラー
