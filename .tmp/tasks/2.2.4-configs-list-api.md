## 概要
設定管理システムの基盤APIとして、`GET /api/v1/configs`エンドポイントを実装する。このAPIは設定項目の一覧を取得する機能を提供し、フロントエンドの設定管理UIやダッシュボード画面で使用される。認証が必要なAPIとして実装し、検索・フィルタリング機能を含む。

## 機能要件

### 設定項目一覧取得
- **目的**: 全設定項目の一覧を取得
- **機能**:
  - 全設定項目の一覧表示
  - システム設定とカスタム設定の区別
  - ページネーション対応
  - 検索・フィルタリング機能

### 認証・セキュリティ
- **目的**: 設定情報の保護
- **機能**:
  - 管理者認証の必須化
  - アクセスログの記録

### ConfigService拡張
```typescript
// src/services/config.ts に追加するメソッド

interface ConfigsListParams {
  page: number;
  limit: number;
  search: string;
  type?: 'string' | 'boolean' | 'number' | 'json';
  system?: boolean;
  sort: 'key' | 'type' | 'created_at' | 'updated_at';
  order: 'asc' | 'desc';
}

interface ConfigsListResult {
  configs: Config[];
  pagination: {
    total: number;
    page: number;
    limit: number;
    totalPages: number;
    hasNext: boolean;
    hasPrev: boolean;
  };
}

/**
 * 設定項目一覧を取得（ページネーション、検索、フィルタリング対応）
 */
static async getConfigsList(params: ConfigsListParams): Promise<ConfigsListResult> {
  const { page, limit, search, type, system, sort, order } = params;
  const offset = (page - 1) * limit;

  // クエリビルダーの構築
  let query = this.db.select().from(configTable);

  // 検索条件の追加
  const conditions = [];
  if (search) {
    conditions.push(
      or(
        like(configTable.key, `%${search}%`),
        like(configTable.description, `%${search}%`)
      )
    );
  }
  if (type) {
    conditions.push(eq(configTable.type, type));
  }
  if (system !== undefined) {
    conditions.push(eq(configTable.system_config, system ? 1 : 0));
  }

  if (conditions.length > 0) {
    query = query.where(and(...conditions));
  }

  // ソート条件の追加
  const sortColumn = {
    'key': configTable.key,
    'type': configTable.type,
    'created_at': configTable.created_at,
    'updated_at': configTable.updated_at
  }[sort];

  query = order === 'desc' 
    ? query.orderBy(desc(sortColumn))
    : query.orderBy(asc(sortColumn));

  // 総件数の取得
  const countQuery = this.db.select({ count: count() }).from(configTable);
  if (conditions.length > 0) {
    countQuery.where(and(...conditions));
  }
  const [{ count: total }] = await countQuery.execute();

  // ページネーション適用
  query = query.limit(limit).offset(offset);

  // データの取得
  const configs = await query.execute();

  // ページネーション情報の計算
  const totalPages = Math.ceil(total / limit);
  const hasNext = page < totalPages;
  const hasPrev = page > 1;

  return {
    configs,
    pagination: {
      total,
      page,
      limit,
      totalPages,
      hasNext,
      hasPrev
    }
  };
}
```


## エラーハンドリング

### エラーケース一覧
1. **認証エラー (401)**
   - 認証トークンが無効または期限切れ
   - 認証ヘッダーが不正

2. **権限エラー (403)**
   - 設定管理権限が不足

3. **バリデーションエラー (400)**
   - 無効なクエリパラメータ
   - ページ番号やリミットの範囲外指定

4. **システムエラー (500)**
   - データベース接続エラー
   - 予期しない内部エラー

### エラーレスポンス例

```typescript
describe('GET /api/v1/configs', () => {

      });
      
      expect(response.status).toBe(200);
      const data = await response.json();
      expect(data.data.configs).toSatisfy(configs => 
        configs.every(config => 
          config.key.includes('google') || 
          (config.description && config.description.includes('google'))

      const response = await app.request('/api/v1/configs', {
        headers: { 'Authorization': `Bearer ${validToken}` }
      });
      
      const data = await response.json();

      expect(secretConfig.type).toBe('string');
    });

  describe('異常系', () => {
    it('未認証ユーザーは401エラーが返る', async () => {

      const response = await app.request('/api/v1/configs?page=0', {

      });
      expect(response.status).toBe(400);
    });
```

```typescript
});

// ハンドラー実装
import { OpenAPIHono } from '@hono/zod-openapi';
import { Context } from 'hono';
import { ConfigService } from '../../../services/config';
import { checkConfigAuthentication } from '../../../utils/auth';

const app = new OpenAPIHono<{ Bindings: Env }>();

app.openapi(getConfigsListRoute, async (c) => {
  try {
    // 認証チェック
    const isAuthenticated = await checkConfigAuthentication(c);
    if (!isAuthenticated) {
      return c.json({
        success: false,
        error: {
          code: 'UNAUTHORIZED',
          message: 'Authentication required'
        }
      }, 401);
    }

    // バリデーション済みクエリパラメータの取得
    const { page, limit, search, type, system, sort, order } = c.req.valid('query');

    // 設定項目の取得
    const result = await ConfigService.getConfigsList({
      page, limit, search, type, system, sort, order
    });
    
    // validationフィールドのパース
    const configs = result.configs.map(config => ({
      ...config,
      validation: (() => {
        try {
          return config.validation ? JSON.parse(config.validation) : null;
        } catch {
          return null;
        }
      })()
    }));

    return c.json({
      success: true,
      data: {
        configs,
        pagination: result.pagination
      }
    });

  } catch (error) {
    console.error('Get configs list error:', error);
    return c.json({
      success: false,
      error: {
        code: 'INTERNAL_ERROR',
        message: 'Failed to retrieve configuration list'
      }
    }, 500);
  }
});

export default app;
```
