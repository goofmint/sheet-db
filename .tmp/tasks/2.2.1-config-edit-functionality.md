## 概要
設定管理UI（`GET /config`）に設定値の編集・保存機能を追加する。管理者が各設定項目を動的に変更し、リアルタイムでアプリケーション設定を更新できる機能を実装する。

## 機能要件

### 設定値編集フォーム
- **目的**: 設定一覧画面で各設定項目を編集可能にする
- **機能**:
  - 設定値のインライン編集
  - データ型に応じた入力フィールド（テキスト、チェックボックス、セレクト、JSONエディタ）
  - リアルタイム保存またはバッチ保存

### 設定保存API
- **エンドポイント**: `POST /api/v1/config/update`
- **目的**: 設定値の更新をAPIで処理
- **機能**:
  - 個別設定項目の更新
  - 複数設定項目の一括更新
  - 設定値のバリデーション

## API Design

### RESTful Endpoints

#### GET /config - Configuration List Display
Display all configuration items in read-only table format with edit/delete icons.

#### GET /config/:key/edit - Edit Configuration Item
Show edit form for specific configuration item.

#### JavaScript Functions

```javascript
// Delete configuration item
async function deleteConfig(key) {
  if (!confirm(`Are you sure you want to delete configuration "${key}"?`)) {
    return;
  }
  
  const response = await fetch(`/api/v1/configs/${encodeURIComponent(key)}`, {
    method: 'DELETE',
    headers: {
      'X-Requested-With': 'XMLHttpRequest'
    }
  });
  
  const result = await response.json();
  if (result.success) {
    // Remove the row from the table
    const row = document.querySelector(`tr[data-key="${key}"]`);
    if (row) {
      row.remove();
    }
    showNotification(true, result.message);
  } else {
    showNotification(false, result.error?.message || 'Failed to delete configuration');
  }
}

// Edit configuration form submission
document.getElementById('config-edit-form')?.addEventListener('submit', async (e) => {
  e.preventDefault();
  
  const formData = new FormData(e.target);
  const key = formData.get('key');
  const requestData = {
    value: formData.get('value'),
    type: formData.get('type'),
    description: formData.get('description') || null
  };
  
  const response = await fetch(`/api/v1/configs/${encodeURIComponent(key)}`, {
    method: 'PUT',
    headers: {
      'Content-Type': 'application/json',
      'X-Requested-With': 'XMLHttpRequest'
    },
    body: JSON.stringify(requestData)
  });
  
  const result = await response.json();
  if (result.success) {
    showNotification(true, result.message);
    // Redirect back to config list after successful update
    setTimeout(() => {
      window.location.href = '/config';
    }, 1500);
  } else {
    showNotification(false, result.error?.message || 'Failed to update configuration');
  }
});

// Dynamic input type switching based on data type selection
document.getElementById('config-type')?.addEventListener('change', function(e) {
  const valueContainer = document.querySelector('.value-input-container');
  const currentValue = document.getElementById('config-value').value;
  const newType = e.target.value;
  
  let newInput;
  switch (newType) {
    case 'boolean':
      newInput = `
        <select id="config-value" name="value" class="form-select" required>
          <option value="true" ${currentValue === 'true' ? 'selected' : ''}>true</option>
          <option value="false" ${currentValue === 'false' ? 'selected' : ''}>false</option>
        </select>
      `;
      break;
    case 'number':
      newInput = `
        <input type="number" 
               id="config-value" 
               name="value" 
               value="${currentValue}"
               class="form-input"
               step="any"
               required>
      `;
      break;
    case 'json':
      newInput = `
        <textarea id="config-value" 
                  name="value" 
                  class="form-textarea json-editor"
                  rows="6"
                  required>${currentValue}</textarea>
      `;
      break;
    default: // string
      newInput = `
        <input type="text" 
               id="config-value" 
               name="value" 
               value="${currentValue}"
               class="form-input"
               required>
      `;
  }
  
  valueContainer.innerHTML = newInput;
});

// Notification display function
function showNotification(success, message) {
  // Create or update notification element
  let notification = document.getElementById('notification');
  if (!notification) {
    notification = document.createElement('div');
    notification.id = 'notification';
    document.body.appendChild(notification);
  }
  
  notification.className = `notification ${success ? 'success' : 'error'}`;
  notification.textContent = message;
  notification.style.display = 'block';
  
  // Auto-hide after 3 seconds
  setTimeout(() => {
    notification.style.display = 'none';
  }, 3000);
}
```

## バリデーション仕様

### 設定項目別バリデーション
```typescript
interface ConfigValidators {
  'google.client_id': (value: string) => boolean;
  'google.client_secret': (value: string) => boolean;
  'auth0.domain': (value: string) => boolean;
  'auth0.client_id': (value: string) => boolean;
  'auth0.client_secret': (value: string) => boolean;
  'app.config_password': (value: string) => boolean;
  'cache.ttl_minutes': (value: string) => boolean;
}

const validators: ConfigValidators = {
  'google.client_id': (value) => value.length > 0 && value.includes('.googleusercontent.com'),
  'google.client_secret': (value) => value.length >= 24,
  'auth0.domain': (value) => /^[a-zA-Z0-9.-]+\.auth0\.com$/.test(value),
  'auth0.client_id': (value) => value.length >= 32,
  'auth0.client_secret': (value) => value.length >= 64,
  'app.config_password': (value) => value.length >= 8,
  'cache.ttl_minutes': (value) => {
    const num = parseInt(value, 10);
    return !isNaN(num) && num > 0 && num <= 1440; // 1日以内
  }
};
```

## セキュリティ要件

### 認証・認可
1. **パスワード認証必須**
   - Cookie: `config_auth` の確認
   - 未認証時は401エラー

2. **CSRF対策**
   - `X-Requested-With: XMLHttpRequest` ヘッダー必須
   - Origin/Refererヘッダーの検証

3. **入力サニタイゼーション**
   - HTMLエスケープ処理
   - SQLインジェクション対策
   - バリデーション必須
